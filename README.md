## Get setup

### Clone the repo

```sh
git clone git@github.com:koacow/harkness.git
cd /harkness/react-site
```

### Frontend setup

1.Prerequisites:

- homebrew
- Node.js
- NPM

2.Install frontend dependencies

```sh
cd frontend
npm i
```

3.Start the app:

```sh
npm run dev
```
